
# 🔐 Login Page Practice (MERN Authentication)

This project is a full-stack authentication system developed using the MERN stack (MongoDB, Express.js, React, and Node.js).
It demonstrates core authentication functionalities such as user registration,
login, and email integration using SMTP services.
---

## 🌐 Live Demo

👉 [Click here to view the app](https://loginpage-practice.onrender.com)


## 🚀 Features

* 🔑 User Registration & Login
* 📧 Email sending using SMTP (Brevo)
* 🔒 Secure environment variable handling
* ⚡ Fast frontend built with React + Vite
* 🌍 Deployed on cloud (Render)


## 🛠️ Tech Stack

### Frontend

* React
* Vite
* CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Other Tools

* Nodemailer
* Brevo SMTP
